# Arunodhaya Cafe Digital Suite
A professional, responsive web solution designed for Arunodhaya Cafe to bridge the gap between traditional dining and digital convenience.

### 🚀 Features
- **QR-Based Menu:** Fast access to digital menus for diners.
- **Dynamic Rating System:** Integrated feedback loop for customer satisfaction.
- **Mobile-First Design:** Optimized for seamless ordering on any device.

### 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, Tailwind CSS
- **Interactions:** JavaScript (ES6+)

### 📈 Business Impact
Streamlined the ordering process and digitized customer feedback, reducing wait times and improving service quality.
